## Color Palette Hints by Industry

- **SaaS**: Neutral base (slate/gray), single accent (blue, indigo, or brand).
- **E-commerce**: Warm neutrals, strong CTA (e.g. orange/red or brand).
- **Fintech**: Blue/green for trust; avoid purple/pink gradients.
- **Healthcare / Wellness**: Soft greens, blues, warm whites; avoid harsh contrast.
- **Beauty / Luxury**: Soft pink, sage, gold accent; elegant serif + clean sans.
- **Creative / Portfolio**: Variable; match brand; avoid generic “AI” gradients.

## Typography Moods

- **Professional / B2B**: Inter, DM Sans, system-ui.
- **Elegant / Luxury**: Cormorant Garamond, Playfair, serif + Montserrat/Open Sans.
- **Friendly / Wellness**: Nunito, Lato, rounded sans.
- **Tech / Data**: JetBrains Mono for data, Inter or Geist for UI.
- **Editorial / Creative**: Distinct serif + neutral sans (e.g. Source Serif + system).
